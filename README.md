# Лабораторна робота 3 (Рівень 3, варіант 2): Збалансоване бінарне дерево

## Опис завдання
Для заданого бінарного дерева необхідно перевірити, чи є воно збалансованим. Бінарне дерево вважається збалансованим, якщо різниця у висоті його лівого та правого піддерев не перевищує 1 для будь-якого піддерева.

## Реалізація
Функція `is_tree_balanced(node: BinaryTree) -> bool` отримує на вхід корінь дерева та повертає `True`, якщо дерево збалансоване, і `False` у протилежному випадку. 

## Файли
- `balanced_tree.py` — клас дерева та функція перевірки балансу.
- `test_balanced_tree.py` — тести для перевірки різних сценаріїв.
- `tree_data.txt` — вхідні дані для побудови дерева.

# Лабораторна робота 4 (Рівень 3, варіант 2): Черга з пріоритетами (AVL-дерево)

## Опис завдання
Реалізувати структуру даних "черга з пріоритетами" на основі AVL-дерева, в якому батьківський елемент має вищий пріоритет, ніж елемент справа, або нижчий чи рівний пріоритет, ніж пріоритет його лівої дитини.

## Підтримувані операції
1. Вставка елемента з заданим значенням та пріоритетом до черги.
2. Видалення та повернення елемента з найвищим пріоритетом з черги.
3. Перегляд черги без її зміни (peek).
4. Додатково: видалення довільного елемента за значенням.

## Файли
- `avl_priority_queue.py` — реалізація класів `Node` та `AVLPriorityQueue`.
- `test_avl_priority_queue.py` — тести структури даних.

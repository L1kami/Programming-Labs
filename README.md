# Лабораторна робота 2: Мінімальна квадратна дошка

## Опис завдання
Компанії NanoSoft потрібно розмістити N прямокутних листків розміром W × H на квадратній дошці. Листки не можна повертати або накладати один на одний. Необхідно знайти мінімальний розмір сторони квадратної дошки, яка здатна вмістити всі ці листки.

## Вхідні та вихідні дані
- **Вхідні дані:** Три цілих числа N, W, H — кількість листків, ширина та висота листка відповідно.
- **Вихідні дані:** Одне число — мінімальна довжина сторони квадратної дошки.

## Файли
- `solution.py` — реалізація алгоритму (бінарний пошук).
- `test_solution.py` — тести.

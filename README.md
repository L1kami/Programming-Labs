# Лабораторна робота 1 (Рівень 2, варіант 2): Система управління портфелем брокера

## Опис завдання
Реалізовано об'єктно-орієнтовану систему для управління фінансовими активами (акції та ф'ючерси з кредитним плечем). Система дозволяє додавати активи, оновлювати їх ціни, частково або повністю продавати, а також виводити детальну статистику прибутковості портфеля.

## Запуск
Головний файл для демонстрації роботи: `main.py`
Тести: `test_lab1.py` (використовується `unittest`).
